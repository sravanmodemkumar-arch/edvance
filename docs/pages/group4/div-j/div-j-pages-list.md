## Purpose

Division J consolidates the management information function for the college portal.
Unlike Division-specific MIS pages (B-07 for academic, E-08 for placement, F-06 for research, etc.), Division J provides:

- **Cross-domain executive dashboards** (Principal, Trust)
- **External submission generators** (NIRF, NAAC, AICTE)
- **Government and regulatory returns** (AICTE, UGC, JNTU, DPIIT, MoE)
- **Governing Body quarterly/annual reports**

All data in Division J is aggregated from source modules (A–I); Division J produces no original data — it only presents, combines, and formats existing data.
