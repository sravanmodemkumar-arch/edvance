## 4. API Endpoints

| # | Method | Endpoint | Description |
|---|---|---|---|
| 1 | `GET` | `/api/v1/student/profile/` | Student profile details (personal info, exam prefs, subscription) |
| 2 | `PUT` | `/api/v1/student/profile/` | Update profile fields (name, phone, exam preferences, photo) |
| 3 | `GET` | `/api/v1/student/profile/performance/trends/` | Score trend data across mock tests (chart-ready) |
| 4 | `GET` | `/api/v1/student/profile/performance/history/?page=1` | Paginated exam history with scores, ranks, percentiles |
| 5 | `GET` | `/api/v1/student/profile/performance/subjects/` | Subject-wise accuracy, question counts, and ranks |
| 6 | `GET` | `/api/v1/student/profile/badges/` | Earned badges and achievements with unlock dates |
| 7 | `GET` | `/api/v1/student/profile/certificates/` | List of issued certificates with download URLs |
| 8 | `GET` | `/api/v1/student/profile/certificates/{cert_id}/download/` | Download certificate as branded PDF |
| 9 | `POST` | `/api/v1/student/profile/performance/export/` | Export full exam history as CSV or PDF report |

## 5. Business Rules

- The student profile page enforces strict data minimisation in compliance with India's Digital Personal Data Protection Act 2023; only data essential for the exam preparation service is collected — name, email, phone, city, age, gender, and exam preferences; sensitive categories like caste, religion, income, and Aadhaar number are never collected even though some government exams ask for these during actual registration; the TSP cannot add custom mandatory fields that collect sensitive data; if a student requests data deletion under DPDP Section 12, their profile is anonymised (name replaced with "Deleted User #XXXXX", email hashed, phone removed) but their test attempt records are retained in anonymised form for the TSP's aggregate analytics; full deletion including test records is possible but requires TSP admin approval because it affects leaderboard integrity
- Performance trends and subject-wise analytics are computed from the student's complete attempt history within the TSP; the score trend chart includes only full-length mock tests for the selected exam (APPSC Group 2 Prelims for Ravi), excluding subject tests and current affairs quizzes, because mixing different test formats on the same chart would produce misleading trend lines; subject-wise accuracy is calculated across all test types (mocks + subject tests + practice sets) because the goal is to measure subject mastery regardless of test format; the "Consistency Index" (82/100 for Ravi) is a proprietary metric that penalises high variance in scores — a student who scores 140, 80, 135, 75 has lower consistency than one who scores 110, 108, 112, 115, even if the former has a higher average; this metric helps students understand that reliable performance matters more than occasional peaks
- Certificates are auto-generated when a student meets predefined criteria configured by the TSP admin; TopRank Academy issues certificates for: scoring in the top 5% of a full-length mock test, achieving rank #1 in any subject test, maintaining a 30-day study streak, and solving 100+ questions in a single day; the certificate PDF is generated server-side using a template engine with the TSP's logo, brand colours (#1A237E header, #FF6F00 accent), the student's name in a calligraphic font, and a QR code that links to a public verification page at verify.toprankacademy.in/cert/{cert_id}; this verification page is accessible without login so that students can share certificates on LinkedIn or WhatsApp and recipients can verify authenticity; the TSP admin can add new certificate criteria and modify existing templates through Division B settings
- The exam history export (CSV or PDF) includes every test the student has attempted with date, test name, score, maximum marks, rank, percentile, time taken, and subject-wise breakdown; this export is designed for two use cases: students who want to track their progress in a personal spreadsheet, and parents who want to verify their child's preparation progress (a common pattern in Indian coaching culture where parents invest Rs.2,499/year and expect visibility into study activity); the CSV format is compatible with Excel and Google Sheets with UTF-8 encoding to handle Telugu characters in test names; the PDF report is a branded TopRank Academy document with charts and tables suitable for printing; the export is rate-limited to 5 requests per day per student to prevent automated scraping; the export includes a timestamp and disclaimer that "This report was generated from TopRank Academy's platform and reflects the student's performance as of {date}"
