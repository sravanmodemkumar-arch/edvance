# Division B — Exam Detail System (Dynamic Template)

> **System:** Single template that renders the full detail page for ANY exam from the DB
> **URL Pattern:** `/exam/{exam-slug}/` — e.g. `/exam/appsc-group-2-2025/`, `/exam/ssc-cgl-2026/`
> **Priority:** P1
> **Data:** `exam` + `conducting_body` + `syllabus_node` + `stage` + `cut_off` + `notification`
