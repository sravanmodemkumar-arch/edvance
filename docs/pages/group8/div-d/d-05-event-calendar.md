## 4. API Endpoints

| # | Method | Endpoint | Description |
|---|---|---|---|
| 1 | `GET` | `/api/v1/parent/communicate/events/` | List events (filterable by child, institution, category, date range) |
| 2 | `GET` | `/api/v1/parent/communicate/events/{id}/` | Get event detail with programme, child participation info |

| 4 | `GET` | `/api/v1/parent/communicate/events/{id}/rsvp/` | Get current RSVP status for an event |
| 5 | `DELETE` | `/api/v1/parent/communicate/events/{id}/rsvp/` | Cancel RSVP |
| 6 | `GET` | `/api/v1/parent/communicate/events/calendar/` | Get events in calendar format (month/week view) |
| 7 | `GET` | `/api/v1/parent/communicate/events/calendar/sync/` | Get .ics feed URL for external calendar sync |
| 8 | `GET` | `/api/v1/parent/communicate/events/upcoming/` | Get next 10 upcoming events across all institutions |

## 5. Business Rules

- The event calendar aggregates events from all institutions linked to the parent's children into a single unified view, resolving the real problem of parents with children in multiple institutions having to check three separate portals or WhatsApp groups for event information; events are colour-coded by institution and categorised by type (academic, cultural, sports, ceremony, holiday, exam, deadline); the calendar supports Google Calendar and Apple Calendar sync via a standard .ics feed URL that auto-updates when new events are added by any institution; this means Mrs. Lakshmi Devi sees GCEH's Sports Day, Sri Chaitanya's Annual Day, and TopRank Academy's EAMCET exam dates all in one view, and all three appear automatically in her phone's calendar app without manual entry.

- RSVP for events serves dual purposes: it helps institutions plan logistics (seating, food, parking) and it creates a commitment mechanism that increases parent attendance at school events; when a parent RSVPs "Yes," they receive a reminder SMS 24 hours before the event and another 2 hours before; the system tracks RSVP-to-attendance conversion (did the parent actually show up, verified via gate check-in QR code) and this data helps institutions estimate realistic turnout for future events; RSVP responses are capped at 3 attendees per family (parent plus 2 guests) for space-constrained events like Annual Day, and the institution can set a maximum RSVP limit after which additional parents are waitlisted; for events where the child is receiving a prize or performing, the RSVP invitation is flagged with a special notification encouraging the parent to attend.

- Events where the child is a participant, performer, or prize recipient are highlighted prominently in the calendar and event detail views; the system pulls participation data from the institution's event management module (which class teacher or event coordinator updates) and maps it to the parent's child; this ensures parents do not miss events that are personally significant вҖ” if Priya is receiving a topper prize at Annual Day, the event card shows "Your child is receiving a prize" in a prominent banner; similarly, if Ravi is playing cricket on Sports Day, the parent sees his team and match schedule; this personalisation transforms the event calendar from a generic institutional notice board into a child-specific engagement tool that makes parents feel connected to their child's institutional life.

- Holiday and exam date events are auto-populated from the institution's academic calendar and cannot be modified by the event coordinator (they are system-generated from the academic calendar module); this prevents discrepancies between the official academic calendar and the event calendar shown to parents; gazetted holidays (Republic Day, Independence Day, Ambedkar Jayanti, Gandhi Jayanti) are pre-loaded for all institutions based on the state government notification; state-specific festivals (Ugadi, Sankranti, Bathukamma for Telangana, Bonalu) are added based on the institution's state registration; exam dates are synced from the examination module and show the child-specific exam timetable (not the full institution exam schedule); when an exam is rescheduled (common during natural disasters or bandh situations in Andhra Pradesh and Telangana), the calendar event is auto-updated and the parent receives an immediate notification about the change with the old and new dates clearly shown.
