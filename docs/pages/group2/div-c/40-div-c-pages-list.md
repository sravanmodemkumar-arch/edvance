# Division C — Group Admissions: Page Index

## Division Overview

Division C covers the complete Group Admissions function for EduForge — every workflow from initial student enquiry to final enrollment confirmation, scholarship award, demo class coordination, alumni engagement, and referral admissions. The division serves seven G3-level roles: Group Admissions Director (Role 23), Group Admission Coordinator (Role 24), Group Admission Counsellor (Role 25), Group Scholarship Exam Manager (Role 26), Group Scholarship Manager (Role 27), Group Alumni Relations Manager (Role 28), and Group Demo Class Coordinator (Role 29). All Division C pages operate under the URL prefix `/group/adm/` and aggregate data from all branches in the group. Branch-level admission operations are managed in branch-level modules; Division C provides the group leadership layer that oversees, coordinates, and reports on those branch operations across the entire institution.

**Total pages:** 42 page specs + 1 index = 43 files.
