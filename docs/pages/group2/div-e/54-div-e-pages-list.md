## Technology Notes

- **Backend:** FastAPI
- **Frontend:** Django + HTMX + Tailwind CSS
- **Database:** PostgreSQL (all metadata and records)
- **File Storage:** Cloudflare R2 (document vault — staff documents, policy PDFs, BGV evidence)
- **Auth:** JWT (role-based access control enforced at API level)
- **Pagination:** Server-side throughout — no client-side pagination
- **Caching:** No Redis — all data fetched from PostgreSQL directly
- **Template base:** `portal_base.html` (all pages)
