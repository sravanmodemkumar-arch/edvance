## 1. Purpose

The Special Ed Progress Reports page provides aggregated, downloadable reporting on the group's special education programme — covering IEP goal achievement, accommodation usage, student-level progress summaries, and incident logs. These reports serve two audiences: internal leadership (CAO and Academic Director reviewing programme effectiveness) and external compliance bodies (NCPCR, CBSE Inclusive Education Cell, state board authorities).

In the Indian regulatory environment, institutions with government affiliations or CBSE recognition are expected to demonstrate active, documented inclusive education practice. The quarterly NCPCR-format report generated by this page provides the evidence trail required for compliance inspections. The IEP goal achievement report allows the Special Education Coordinator to present data-backed progress to school management and to parents at annual IEP review meetings.

All reports generated from this page are DPDP Act 2023 compliant by design: individual student identifiers are included only in reports accessed by authorised roles (G3+ Special Ed Coord, G4 CAO), while the MIS Officer can only download anonymised branch-level summaries. Reports are watermarked with the accessor's name and timestamp to discourage unauthorised sharing.

## 4. Main Content

### 4.1 Report Builder (primary action area)

**Step 1 — Select Report Type:**

| Report Type | Description | Student-identified | Roles with access |
|---|---|---|---|
| Branch Summary | Aggregated per-branch: student count, IEP count, accommodation count, avg goal achievement % | No | All roles on this page |
| IEP Goal Achievement | Per-student IEP progress — goals met %, trend over reviews | Yes | Spec Ed Coord, CAO, Academic Dir |
| Accommodation Usage | Types of accommodations approved, by exam, by branch | No | All roles |
| Incident Log | POCSO / welfare incidents flagged from branch level | Yes (student ID only) | Spec Ed Coord, CAO |
| NCPCR Compliance Format | Quarterly statutory report in NCPCR prescribed format | Partially (anonymised by student ID) | Spec Ed Coord, CAO |

**Step 2 — Parameters:**

| Parameter | Type | Required | Notes |
|---|---|---|---|
| Branch scope | Multi-select / All | ✅ | All branches or specific |
| Term / Date range | Select / Date range | ✅ | Current term pre-selected |
| Academic Year | Select | ✅ | |

**Step 3 — Preview & Export:**
- [Preview Report] — renders inline HTML preview below the builder
- [Download PDF] — PDF with group letterhead; watermarked with accessor name + timestamp
- [Download XLSX] — raw data for further analysis (student-identified reports: access logged)
