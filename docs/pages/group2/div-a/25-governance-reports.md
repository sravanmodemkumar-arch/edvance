# 25 — Governance Reports

> **URL:** `/group/gov/reports/`
> **File:** `25-governance-reports.md`
> **Template:** `portal_base.html` (light theme)
> **Priority:** P1
> **Roles:** Chairman G5 · MD G5 · CEO G4 (full) · President G4 (Academic) · VP G4 (Ops) · Trustee G1 (read) · Exec Secretary G3 (send only) · Advisor G1 (read)

---

## 1. Purpose

Build, schedule, and manage governance reports for the institution group. Reports are generated
from live platform data — no manual compilation. Reports can be scheduled to auto-generate and
email to recipients on a fixed schedule.

Report types: Monthly MIS · Financial Summary · Academic Performance · Compliance Status · Board Pack.

The Executive Secretary can send generated reports; the Chairman/MD/CEO generate and schedule them.

## 4. Generated Reports Tab

**Search:** Report name. Debounce 300ms.

**Filters:** Type · Generated by · Date range.

**Columns:**

## 7. Drawers & Modals

### 7.1 Drawer: `report-builder`
- **Trigger:** [+ Build Report] or Template card → [Build Report]
- **Width:** 760px
- **Tabs:** Sections · Date Range · Branches · Recipients · Schedule

#### Tab: Sections
- Checklist of available sections (based on selected template or custom)
- Each section: checkbox + preview text describing what data it includes
- Reorder sections via drag handle (order = order in final PDF)

#### Tab: Date Range
| Field | Type | Required | Validation |
|---|---|---|---|
| Report Period | Select | ✅ | This Month · Last Month · This Quarter · Last Quarter · This Year · Custom |
| Custom Start | Date | Conditional | |
| Custom End | Date | Conditional | |
| Compare With | Select | ❌ | None · Previous Period · Same Period Last Year |

#### Tab: Branches
| Field | Type | Required | Validation |
|---|---|---|---|
| Include Branches | Radio | ✅ | All Branches · Select Branches |
| Select Branches | Multi-select | Conditional | |
| Include Zones | Toggle | ❌ | Adds zone-level summary |

#### Tab: Recipients
| Field | Type | Required | Validation |
|---|---|---|---|
| Send To | Multi-select | ❌ | Search by name or email |
| Add External Emails | Text (multi) | ❌ | Comma-separated external emails |
| Subject | Text | ❌ | Email subject line |
| Message | Textarea | ❌ | Accompanying message |

#### Tab: Schedule
| Field | Type | Required | Validation |
|---|---|---|---|
| Generate | Radio | ✅ | Now · On Schedule |
| Frequency (if scheduled) | Select | Conditional | Daily · Weekly (day of week) · Monthly (day of month) · Quarterly |
| Start Date | Date | Conditional | |
| End Date | Date | ❌ | Optional schedule end |

**Submit:** [Generate Report] or [Save Schedule] — shows progress.

**Generation:** Async — "Report generating in background. You'll be notified when ready." Toast.

### 7.2 Drawer: `schedule-report`
- Same as `report-builder` but Schedule tab is primary and Generate Now is hidden.

### 7.3 Modal: `send-report`
- **Trigger:** Generated Reports → [Send]
- **Width:** 480px
- **Fields:** Recipients (multi-select from provisioned users + additional emails) · Subject (pre-filled) · Message (optional) · Attach PDF (toggle, default on)
- **Buttons:** [Send Report] + [Cancel]

## 13. API Endpoints

| Method | Endpoint | Auth | Description |
|---|---|---|---|
| GET | `/api/v1/group/{id}/reports/` | JWT | Generated reports list |
| POST | `/api/v1/group/{id}/reports/generate/` | JWT (G4/G5) | Trigger report generation |
| GET | `/api/v1/group/{id}/reports/{rid}/` | JWT | Report detail |

| DELETE | `/api/v1/group/{id}/reports/{rid}/` | JWT (G5) | Delete report |
| GET | `/api/v1/group/{id}/reports/scheduled/` | JWT | Scheduled reports |
| POST | `/api/v1/group/{id}/reports/scheduled/` | JWT (G4/G5) | Create schedule |
| PUT | `/api/v1/group/{id}/reports/scheduled/{sid}/` | JWT (G4/G5) | Update schedule |
| DELETE | `/api/v1/group/{id}/reports/scheduled/{sid}/` | JWT (G4/G5) | Delete schedule |
| GET | `/api/v1/group/{id}/reports/templates/` | JWT | Available templates |
