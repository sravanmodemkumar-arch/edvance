# 31 — Operations MIS Report

> **URL:** `/group/ops/reports/mis/`
> **File:** `31-operations-mis-report.md`
> **Template:** `portal_base.html` (light theme)
> **Priority:** P1
> **Roles:** COO G4 (full — generate + schedule) · Operations Manager G3 (view + schedule) · Zone Director G4 (zone MIS)

---

## 1. Purpose

Auto-generated monthly Management Information System (MIS) report covering all operational
metrics across branches. Sent to COO, CEO, and Chairman. Provides a structured snapshot
for executive review without requiring manual report compilation.

---

## 2. Report Sections

| Section | Contents | Data Source |
|---|---|---|
| 1. Executive Summary | Key metrics, trends, critical alerts | All ops data |
| 2. Branch Operations | SLA compliance per branch, escalations | Pages 07–08 |
| 3. Grievance Resolution | Open, resolved, overdue, category breakdown | Page 11 |
| 4. Escalations | By severity, type, resolution time | Page 12 |
| 5. Coordinator Activity | Visits completed, issues raised | Pages 09–10 |
| 6. Procurement | Spend vs budget, active POs, delayed deliveries | Pages 18–22 |
| 7. Facilities / Maintenance | Open tickets, resolved, CAPEX status | Pages 25–28 |
| 8. Safety Compliance | Expiring/expired certificates per branch | Page 30 |
| 9. Zone Performance (large) | Zone-by-zone operational KPIs | Pages 15–17 |
| 10. Operational Compliance | Branch checklist scores | Page 13 |

## 5. Report Preview

**Layout:** Full rendered report preview in page (not a PDF iframe — Django template rendered in page).

Sections collapsible. Each section shows:
- Section title + key metric highlights (bold)
- Charts (inline, Chart.js rendered)
- Data tables (compact, read-only)
- Narrative summary (auto-generated: "Branch X saw SLA decline from 92% to 84% — 3 escalations raised")

---

## 6. Report History Table

**Columns:** Month/Year · Generated Date · Generated By · Recipients · Download PDF · Actions (View · Resend)

**Pagination:** 12/page (12 months = 1 year on one page).

---

## 7. Generate Report Drawer

- **Width:** 480px
- **Trigger:** [Generate Now]
- **Fields:**
  - Month (pre-filled to current month)
  - Sections to include (multi-select checkboxes, all selected by default)
  - Zone scope (All / Specific zone — for zone-level MIS)
  - Include charts: Yes/No
- **[Generate Report]:** API call → generates report → notifies when ready (< 30 seconds)

---

## 8. Schedule Report Drawer

- **Width:** 480px
- **Fields:**
  - Frequency: Monthly (default) · Quarterly
  - Day of month: 1st to 5th
  - Recipients: COO (always) + add others (CEO, Chairman, Ops Manager, etc.)
  - Delivery channel: Email + EduForge notification + WhatsApp
  - Active: toggle
