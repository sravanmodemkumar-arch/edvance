# Home Page: Group 6 — Exam Domain Portal
**Route:** `/home`
**Domain:** `ssc.eduforge.in` / `rrb.eduforge.in` / `upsc.eduforge.in` etc.
**Access:** Self-registered aspirants — Free tier (5 tests/month) and Premium subscribers
**Portal type:** Public exam domain — SSC, RRB, UPSC, Banking, AP Board, TS Board

---

## Overview

| Property | Value |
|---|---|
| Purpose | National mock test platform for competitive exam aspirants |
| Who uses it | Self-registered students aged 18+ preparing for SSC CGL, RRB NTPC, UPSC, Banking etc. |
| Free vs Premium | Free: 5 tests/month. Premium: unlimited tests + AI plan + advanced analytics |
| Key feature | ALL-India Rank (AIR) among lakhs of aspirants on every test |
| No institution | These are standalone users — not enrolled in any school/coaching |

## API Calls

| Section | Endpoint |
|---|---|
| KPI Bar | `GET /api/v1/domain/home/kpis` |

| Performance charts | `GET /api/v1/student/performance?period=3m` |
| Leaderboard | `GET /api/v1/domain/leaderboard?test_id={}` |
| AI study plan | `GET /api/v1/student/ai-plan` |
| Subscription status | `GET /api/v1/student/subscription` |
