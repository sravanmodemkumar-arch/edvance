# EduForge — Group 6: Online Exam Domain Roles

> Covers all competitive exam preparation domains on EduForge.
> Scale: 6 domains · 5,000 students each at launch · unlimited scale target.
> Domains: SSC · RRB · AP State Board · TS State Board · UPSC · Banking.

---

## What Exam Domains Are — Different From Everything Else

| Aspect | School / College / Coaching | Exam Domain |
|---|---|---|
| Physical location | Campus / classroom | 100% online — no campus |
| Students | Local / city level | All India — any student anywhere |
| Teacher-student bond | Personal, daily | Anonymous — content-driven |
| Admissions | Counsellor-driven | Self-signup — pay online instantly |
| Batches | Fixed time, fixed group | Unlimited — student picks their pace |
| Revenue | Annual fee per institution | Per subscription — B2C + B2B |
| Content | Taught live | Pre-built MCQ bank + test series |
| Results | Board / internal exam | Government exam selection rate |
| Competition | Local coaching | Compete with Testbook, PW, Unacademy |
| Scale | Hundreds per batch | Millions simultaneously |
| BGV / POCSO | Mandatory (minors on campus) | Minimal (online adult users) |

## Exam Domain vs All Other Groups

| Aspect | School/College/Coaching | Exam Domain |
|---|---|---|
| Physical presence | Required | Zero |
| Content creation | Teachers / faculty | SMEs + AI pipeline |
| Student age | Minors + adults | Mostly adults (18–30) |
| POCSO obligation | Mandatory | Minimal (no minors on campus) |
| Scale | City / state level | National — all India |
| Concurrency | Hundreds per batch | Lakhs simultaneously |
| Pricing model | Annual institutional fee | Subscription (monthly/annual) |
| B2B channel | Group → branch | Coaching / school buys bulk seats |
| SEO importance | Low | Critical — organic acquisition |
| YouTube | Optional | Core acquisition channel |
| Anti-cheating | Invigilation | AI-based online proctoring |
| Content IP | Internal use | Public product — must protect |
